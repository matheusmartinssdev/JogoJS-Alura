# JogoJS-Alura
Projeto com JavaScript, o jogo do número secreto. Nele, a pessoa usuária precisará inserir alguns números para adivinhar qual foi o número aleatório gerado pelo computador.  Guilherme: Para desenvolver esse jogo, trabalhei com variáveis, loops, condicionais, if e else. Conceitos usados no mercado de trabalho.
